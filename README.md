### 🍉项目简介

项目主要实现通过读取配置文件连接`Oracle`或`MySQL`数据库，根据输入的动态`SQL`脚本展现数据结果集，

并将结果集导出到excel。项目通过JavaFx技术提供界面操作，对普通用户使用非常友好。
